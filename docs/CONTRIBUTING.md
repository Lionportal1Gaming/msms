# Contributing

## Workflow

- Use short-lived branches off `main`.
- Follow Conventional Commits.
- Add or update tests for behavioral changes.
- Update docs when interfaces or operations change.

## Local Checks

- `npm run lint`
- `npm run test`
- `npm run check`
- `cargo test --manifest-path src-tauri/Cargo.toml`

## Pull Requests

- Explain user-visible behavior changes.
- Call out migration or release impacts.
- Link to ADRs when making architectural shifts.
